## 安装Python包

Python库的安装方式: 

 ```bash
pip install + 包名
 ```

举例: 

```bash
pip install numpy
```



需要安装的包的名称列表, 

* numpy
* matplotlib
* panda
* scipy
* opencv-python
* opencv-contrib-python
* PyFaceDet
* PySerial
* pyyaml
* PySide2 上位机
